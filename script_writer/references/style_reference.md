# Style Reference: "The AI Cheese Pie" Style (AI 芝士派)

Based on the "OpenAI Codex Desktop" announcement.

## 1. Structure Breakdown

The script follows a highly effective "Hook -> Value -> Proof -> Expansion -> CTA" structure:

1.  **The Hook (Golden 3 Seconds)**:
    *   **Technique**: "Big News" + "Free" + "FOMO" (Fear Of Missing Out).
    *   **Example**: "OpenAI 这次真的大放血！... 宣布限时免费！... 免费白嫖 GPT-5.2..."
    *   **Key Elements**: Brand name, emotional trigger ("大放血"), clear benefit ("限时免费", "白嫖").

2.  **Reframing / Value Proposition**:
    *   **Technique**: Redefine the tool for the *user's* life, not just its technical function.
    *   **Example**: "它可不简单是一个代码工具，它是你的24小时全自动数字助理。"
    *   **Key Elements**: Contrast ("不简单是... 而是..."), relatable use cases ("写日报", "拉取资讯").

3.  **Low Barrier to Entry (The "Skill" Concept)**:
    *   **Technique**: Analogy to familiar concepts (App Store) to reduce friction.
    *   **Example**: "像逛 App Store 一样点击安装... '帮我装个处理 PDF 的skill'。"
    *   **Key Elements**: "No command line", "Lazy friendly", natural language interaction.

4.  **Target Audience Address (Non-Tech Focus)**:
    *   **Technique**: Directly addressing the "fear" of coding and the specific audience ("Liberal Arts students/文科生").
    *   **Example**: "专为非技术党设计的 GUI 界面... 让普通人头皮发麻的黑底白字..."
    *   **Key Elements**: Empathy, visual contrast (GUI vs Command Line).

5.  **Advanced Feature as Metaphor ("Work Tree")**:
    *   **Technique**: Simplifying a complex feature ("Work Tree" / isolation) using a sci-fi/pop culture metaphor.
    *   **Example**: "平行宇宙... 互相隔离、互不干扰。"
    *   **Key Elements**: "Clones" (分身), "Parallel Universe" (平行宇宙).

6.  **Urgency & Call to Action (CTA)**:
    *   **Technique**: Reiterate scarcity and broad appeal.
    *   **Example**: "限时免费阶段... 不管你是不是程序员... 强烈建议下载。"

7.  **Standard Outro**:
    *   **Example**: "我是 AI 芝士派，关注我，带你挖掘更多好用的 AI 工具，下期见！"

## 2. Tone & Voice Characteristics

*   **Enthusiastic & High Energy**: Uses strong emotional words ("大放血", "杀疯了", "头皮发麻").
*   **Colloquial & Internet Slang**: Uses terms like "白嫖" (free ride/get for free), "文科生" (liberal arts students/non-techies), "小白" (newbie implied).
*   **Empathetic**: Understands user pain points (fear of coding, boring repetitive tasks).
*   **Authoritative yet Accessible**: "Not just a code tool" (Authority) -> "Like an App Store" (Accessibility).
*   **Visual**: Uses descriptive language that creates images ("Black background white text", "Parallel universe").

## 3. Formatting For Script

*   Short paragraphs.
*   Clear emphasis (bolding or framing).
*   Direct address ("You", "Your").
