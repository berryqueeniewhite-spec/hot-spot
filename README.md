# AI Hot Spot Writer (AI 热点脚本助手)

这是一个自动化的 AI 内容创作系统，旨在帮助创作者快速挖掘全球最新的科技/AI 热点，并生成高质量的口播短视频脚本。

项目包含两个核心能力：**智能选题挖掘** 和 **风格化脚本写作**，并通过工作流将二者无缝串联。

## 🚀 核心功能

### 1. 选题挖掘助手 (Topic Selector)
*   **功能**：全网搜寻并筛选当前最具传播价值的 AI/科技选题。
*   **数据源**：实时聚合 Google News, TechCrunch, Reddit (r/LocalLLaMA等), X (Twitter) 等平台数据。
*   **筛选标准**：
    *   🕒 **时效性**：严格限制在 **72小时** 内的 Breaking News。
    *   🔥 **爆款指数**：基于实操性、新颖性、视觉潜力等维度打分，只推荐高分选题。

### 2. 脚本创作助手 (Script Writer)
*   **角色设定**：资深科技评论员 "**AI芝士派**"。
*   **写作风格**：
    *   🗣 **极致口语化**：拒绝书面语，模拟真实交谈感。
    *   🚫 **拒绝标题党**：严禁使用恐吓性夸张词汇，主打专业与真诚。
    *   💎 **价值导向**：结构化输出（黄金3秒 -> 核心干货 -> 场景演示 -> 价值升华）。
*   **输出内容**：包含完整的逐字稿及小红书风格标题。

## 🛠 使用方法 (Workflow)

本项目内置了自动化工作流，可一键执行从“找选题”到“写脚本”的全过程。

**启动指令**：
在对话框中输入：
> “开始写今天的脚本” 或 “/tech_news_workflow”

**工作流步骤**：
1.  **自动挖掘**：系统搜索全球热点，按爆款指数排序，生成 Top 10 选题列表。
2.  **人工决策**：用户输入序号选择心仪选题（可补充额外观点）。
3.  **智能写作**：系统自动调用 `Script Writer` 生成初稿。
4.  **润色优化**：支持进一步通俗化改写或调整语气。

## 📂 项目结构

*   `topic_selector/`: 选题挖掘 Skill 配置与规则。
*   `script_writer/`: 脚本写作 Skill 提示词与角色设定。
*   `.agent/workflows/`: 自动化工作流定义文件。
*   `scripts/`: 存放生成脚本的历史记录 (可选)。
