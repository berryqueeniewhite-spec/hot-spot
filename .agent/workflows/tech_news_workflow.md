# 全自动热点脚本创作流 (Auto Tech Writer Flow)

这个工作流旨在将“选题挖掘”和“脚本写作”无缝连接，极大提高效率。

## 步骤详解

1.  **启动挖掘 (Scout)**
    *   Agent 自动调用 `search_web` 搜索今日全球 AI/科技热点 (TechCrunch, X, Google Trends)。
    *   Agent 依据 `topic_selector` 的标准（**必须严格核实 72小时时效性**），提炼出 **Top 10 候选选题**。

2.  **人工决策与投喂 (Human Selection & Context)**
    *   Agent 向用户展示这 10 个选题（含一句话推荐理由）。
    *   Agent 询问：“老板，选哪个？（回复序号）。**如果有指定的参考文章或想强调的观点，请一起发给我**。”

3.  **智能写作 (Drafting)**
    *   Agent 根据用户选择的选题 + 用户提供的补充信息（如有）进行脚本初稿撰写。
    *   调用 `script_writer` 生成初稿。

4.  **润色优化 (Polish)**
    *   Agent 展示初稿后询问：“这版怎么样？需要更通俗一点吗？”
    *   如果用户觉得不够味，Agent 将执行**通俗化重写**：
        *   把专业术语换成生活比喻。
        *   把长句拆成短句。
        *   增加语气词和互动感。


## 使用方法
直接在对话框输入：“**开始写今天的脚本**” 或 “**跑一下工作流**”。
